## API Reference

#### Auth Register

```http
  POST /api/register
```

| Parameter         | Type     | Description   |
| :---------------- | :------- | :------------ |
| `fullName`        | `string` | **Required**. |
| `email`           | `string` | **Required**. |
| `username`        | `string` | **Required**. |
| `password`        | `string` | **Required**. |
| `confirmPassword` | `string` | **Required**. |
